Diago's Electronics & Gadgets

A modern and responsive electronics storefront built with HTML, CSS, and JavaScript, and hosted for free using GitHub Pages.

Features

- Responsive design for mobile and desktop
- Home page with featured products
- Product listing page
- Shopping cart using browser local storage
- WhatsApp checkout integration
- Contact page
- Easy product customization

Project Structure

Diagos-Electronics/

│── css/

│   ├── products.js
│   └── cart.js

How to Use

1. Upload all files to a GitHub repository.
2. Enable GitHub Pages:
   - Go to Settings
   - Select Pages
   - Choose the "main" branch and "/ (root)" folder
3. Save the settings and wait a few minutes.
4. Your website will be live at:

https://yourusername.github.io/Diagos-Electronics/

Customization

- Edit product information in "js/products.js"
- Change colors and layout in "css/style.css"
- Update contact details and WhatsApp number in "contact.html" and "cart.js"
